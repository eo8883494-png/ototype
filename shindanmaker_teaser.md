# オトタイプ かんたん版(しんだんめーかー「分岐診断」用台本)

診断タイプ: **分岐診断**(https://shindanmaker.com/create → 分岐診断をつくる)
タイトル案: 「音楽タイプ診断(かんたん版)〜あなたの音楽の聴き方は16タイプのどれ？」
説明文案: 「4つの質問で、あなたの"音楽の聴き方"タイプが分かります。本格版(20問・友達との相性診断つき)は最後のリンクから。」

## 分岐の流れ(はい/いいえ×4問→16タイプ)

```
Q1「音楽は、大人数で盛り上がってこそ好きになれる方？」
├ はい → F(みんな)
└ いいえ → R(ひとり)

Q2「曲を聴くとき、最初に刺さるのは歌詞の方？」
├ はい → L(歌詞)
└ いいえ → S(音・メロディ)

Q3「好きになったら、とことん熱狂する方？」
├ はい → D(熱狂)
└ いいえ → E(ゆるり)

Q4「みんな知ってる王道の曲の方が好き？」
├ はい → T(王道)
└ いいえ → M(開拓)
```

4問の答えの組み合わせ(F/R × L/S × D/E × T/M)で下記16パターンのどれかに到達する。

## 結果テキスト(16タイプ共通フォーマット)

各結果の末尾に必ずこの一文を入れる:
「本格版(20問の本診断+友達との相性診断)はこちら→ https://eo8883494-png.github.io/ototype/」

- **FLDT フェス感涙型**「サビは、みんなで泣くもの」合唱・感動・王道。みんなで歌うサビで気づけば泣いてるタイプ。
- **FLDM 弾き語り布教型**「剥き出しの言葉を、みんなに配りたい」弾き語り・熱量・開拓。刺さった歌詞を全力で布教するタイプ。
- **FLET カラオケ主役型**「歌える一曲を、いつも持ってる」カラオケ・共感・定番。みんなで盛り上がる鉄板曲を持ってるタイプ。
- **FLEM 名曲布教型**「知る人ぞ知る一曲を、そっと配る」布教・歌詞・隠れた名曲。良い歌詞の曲をそっと薦めるタイプ。
- **FSDT フェス大合唱型**「音は、みんなで浴びるもの」合唱・開放感・夏。大音量をみんなで浴びるのが好きなタイプ。
- **FSDM ライブハウス魂型**「爆音は、浄化」爆音・衝動・ライブ。爆音のライブハウスで浄化されるタイプ。
- **FSET 現場ノリ最優先型**「細かいことは、踊ってから」ノリ・ダンス・トレンド。理屈より先に体が動くタイプ。
- **FSEM フェス回遊型**「無名の神ステージを探す」回遊・発掘・グルーヴ。フェスで無名の当たりステージを探すタイプ。
- **RLDT 歌詞刺さり型**「一行に、殴られにいく」歌詞・共感・没入。刺さる一行を求めて音楽を聴くタイプ。
- **RLDM 世界観考察型**「歌詞の裏まで、読みにいく」考察・世界観・歌詞。歌詞の意味を深読みするタイプ。
- **RLET ドラマ主題歌型**「物語ごと、曲を好きになる」主題歌・物語・定番。物語とセットで曲を好きになるタイプ。
- **RLEM 深夜ポエトリー型**「夜だけ、言葉の解像度が上がる」深夜・詩・開拓。夜に一人で言葉を噛みしめるタイプ。
- **RSDT イヤホン没入型**「音の中に、部屋がある」没入・世界観・ひとり時間。イヤホンの中に自分の部屋を持つタイプ。
- **RSDM サブスク潜水型**「再生数の海を、ひとりで泳ぐ」発掘・サウンド・ネット。再生数の少ない曲を一人で発掘するタイプ。
- **RSET チル・BGM型**「BGMこそ、人生の主役」BGM・作業・心地よさ。作業中のBGMを大事にするタイプ。
- **RSEM 洋楽アンテナ型**「国境より、好みで聴く」洋楽・センス・ボーダレス。国内外関係なく好みで聴くタイプ。
