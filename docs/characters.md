# オトタイプ キャラクター仕様書（全16体）

キャラデザ制作用の資料。1体ごとのブロックをそのままコピペで使えます。
サイトの実装は `chars.mjs`（SVG自動生成）。完成イラストと差し替える場合は 正方形・背景透過(PNG/SVG)・表情が小さくても読めるサイズ感 を推奨。

## 共通仕様（全キャラ共通）

- 体型: 丸いブロブ（まんまるに近い楕円）。輪郭線なし・フラット塗り
- 顔: 目2つ+口+白の丸ほっぺ（左右）
- 足元: 薄い楕円の影
- 頭身: 1頭身（体=顔）
- トーン: かわいい・ゆるい・パステル世界(クリーム/セージ背景)に乗る

## パーツの法則（4軸 → 見た目）

| 軸 | 記号 | 見た目 |
|---|---|---|
| 場所感 | **F**(みんな/現場) | 両腕を上げて全開ポーズ |
| 場所感 | **R**(ひとり/自室) | ヘッドホン装着（腕は下に小さく） |
| 刺さる核 | **L**(歌詞) | 白いふきだし（中に文字線2本）を横に浮かべる |
| 刺さる核 | **S**(音) | 8分音符を横に浮かべる |
| のめり込み | **D**(熱狂) | キラキラの丸目（ハイライト入り）＋大きく開けた口（舌見え） |
| のめり込み | **E**(ゆるり) | にっこり閉じ目（アーチ）＋小さな笑み |
| 探し方 | **T**(王道) | 金の王冠（#F6C453・縁#DCA33B） |
| 探し方 | **M**(開拓) | 緑の探検帽（ドーム#A5CBA0・つば#8FB98B・バンド#7DA878） |

## 色の4系統（グループ共通の世界観）

| 系統 | グループ | タイプ | 雰囲気 |
|---|---|---|---|
| 🟠 サンセットオレンジ系 | 現場×歌詞(FL) | FLDT / FLDM / FLET / FLEM | 夕暮れのフェス・エモ・あたたかい |
| ❤️ ヒートレッド系 | 現場×音(FS) | FSDT / FSDM / FSET / FSEM | 熱・汗・ライブの照明 |
| 💜 ミッドナイトパープル系 | ひとり×歌詞(RL) | RLDT / RLDM / RLET / RLEM | 夜・詩・世界観 |
| 💙 ディープブルー系 | ひとり×音(RS) | RSDT / RSDM / RSET / RSEM | 深海・没入・クール |

共通アクセント色: 王冠=#F6C453 / 探検帽=#A5CBA0 / ヘッドホン=#576661 / 音符=#565D68 / ふきだし=#FFFFFF(線#B9AFA0)

## 🟠 サンセットオレンジ系（現場×歌詞）

### FLDT フェス感涙型
- メインカラー: **#FF7A3C**（明るいオレンジ）
- キャッチ: サビは、みんなで泣くもの

- 頭: 金の王冠
- 持ち物: 歌詞のふきだし
